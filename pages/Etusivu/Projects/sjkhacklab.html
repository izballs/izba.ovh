• Seinäjoki Hacklab RY

    PHP / Javascript / CSS / SQL

    MySQL / nginx / PHP7.3

    Seinajoki.Hacklab.fi

    Seinäjoki Hacklab RY:n kotisivut luotiin yhdistyksen tarpeiden mukaan. Erillisiä kirjastoja on käytössä javascriptin puolella ainoastaan highlight.js, jolla saadaan koodit kauniisti esitettyä. PHP puolella on käytössä SimpleXML, Tidy ja PHPMailer. Hallituksen jäsenet pystyvät myös lähettämään viestit suoraa sivuston kautta Telegram-Irc-Discord siltaan samalla kun lisäävät etusivulle ilmoituksen. Sama ilmoitus myös lähtee käyttäjien sähköpostiosoitteisiin. Palvelimen sponsoroi UpCloud ja hoidin palvelimen asennuksen sähköpostinpalvelua myöden. Sähköpostin välitykseen käytössä on mailjet.
    Ylläpidän myös yhdistyksen Telegram-IRC-Discord silta botteja.
